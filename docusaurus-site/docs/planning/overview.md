## Constitutional Principles

All work governed by 6 core principles:

1. **Primary Source Verification** - All claims traceable to verifiable sources
2. **Citation Integrity** - APA 7th edition format mandatory
3. **Peer-Reviewed Priority** - ‚â•50% peer-reviewed sources (‚â•8 of 15)
4. **Zero Plagiarism Tolerance** - 0% similarity score required
5. **Writing Clarity** - Flesch-Kincaid grade 10-12 target
6. **Reproducibility** - All results reproducible with cited methodology

## Quality Gates

### Gate 1: Source Verification
- [ ] Every claim traced to source
- [ ] All sources accessible
- [ ] ‚â•8 peer-reviewed sources

### Gate 2: Citation Compliance
- [ ] APA 7th edition format

- [ ] 100% citation-reference match
- [ ] All URLs/DOIs functional

### Gate 3: Plagiarism Check
- [ ] 0% similarity (excluding citations)
- [ ] Proper quotation marks and attribution

### Gate 4: Writing Quality
- [ ] FK grade 10-12
- [ ] All terms defined on first use
- [ ] 5,000-7,000 words
- [ ] Grammar and spelling checked

### Gate 5: Reproducibility
- [ ] Methods documented for replication
- [ ] Claims linked to specific source locations
